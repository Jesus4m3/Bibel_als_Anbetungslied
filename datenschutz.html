2. Bereitstellung der Website und Hosting

Diese Website wird als statische Website über GitHub Pages bereitgestellt und ist unter einer eigenen Domain erreichbar. Beim Aufruf der Website werden technisch erforderliche Verbindungsdaten verarbeitet, insbesondere:

  • IP-Adresse

  • aufgerufene Datei bzw. Seite
  • Informationen zu Browser, Betriebssystem und Endgerät
  • Referrer-URL
  • Statuscodes und übertragene Datenmengen

Die Verarbeitung erfolgt, um die Website technisch bereitzustellen, die Stabilität und Sicherheit des Angebots zu gewährleisten und Missbrauch zu verhindern. Rechtsgrundlage ist Art. 6 Abs. 1 lit. f DSGVO.

Laut GitHub-Dokumentation wird beim Besuch einer GitHub-Pages-Website die IP-Adresse der Besucherinnen und Besucher aus Sicherheitsgründen protokolliert. Ergänzend gelten die Datenschutzinformationen von GitHub.

Empfänger der Daten ist der Hosting-Anbieter GitHub. Nähere Informationen zur Datenverarbeitung durch GitHub finden Sie in den Datenschutzhinweisen von GitHub: GitHub Privacy Statement.

Soweit bei GitHub eine Verarbeitung in Drittländern, insbesondere in den USA, erfolgt, verweist GitHub auf geeignete Garantien wie Standardvertragsklauseln sowie auf eine Zertifizierung nach dem EU-U.S. Data Privacy Framework.

Auf die konkrete Speicherdauer serverseitiger Logdaten bei GitHub habe ich keinen vollständigen Einfluss. Maßgeblich sind insoweit die Datenschutzinformationen von GitHub.
