# Erlebe den Osterweg

Dieses Repository enthält die Website zum Musikprojekt **„Erlebe den Osterweg“**  
von **Jens Rentmeister / Jesus4m3 Musik**.

## Worum geht es?

„Erlebe den Osterweg“ erzählt die Ostergeschichte als Anbetungslieder.  
Jedes Lied greift eine biblische Station auf und verbindet Bibeltext, Musik und Glaubensimpulse.

## Inhalte der Website

- Startseite
- Übersichtsseite „Erlebe den Osterweg“
- Liedseiten mit:
  - vollständigem Liedtext
  - Vorschaubild
  - Direktlinks zu Plattformen (z. B. Spotify, Apple Music, iTunes, Deezer, Amazon Music, YouTube Music)
  - YouTube-Video-Link
- Impressum
- Datenschutz

## Technischer Aufbau

- Statische GitHub-Pages-Seite (HTML/CSS/JS)
- Gemeinsames Styling über `css/site.css`
- Mobiles Hamburger-Menü über `js/menu.js`
- Linke Navigation mit eingerückten Unterseiten
- Feste untere Navigation für „Vorherige / Nächste“
