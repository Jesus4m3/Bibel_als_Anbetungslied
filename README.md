# Bibel als Anbetungslied

Dieses Repository enthält eine einfache GitHub-Page (`index.html`).

## GitHub Pages aktivieren

1. Repository auf GitHub öffnen.
2. Zu `Settings` -> `Pages` gehen.
3. Bei **Build and deployment**:
   - `Source`: **Deploy from a branch**
   - `Branch`: **main** (oder dein Standard-Branch), Folder: **/(root)**
4. Speichern.

Danach ist die Seite unter folgender URL erreichbar:

`https://<dein-github-name>.github.io/Bibel_als_Anbetungslied/`
